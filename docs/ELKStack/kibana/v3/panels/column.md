# column

状态：稳定

这是一个伪面板。目的是让你在一列中添加多个其他面板。虽然 column 面板状态是稳定，它的限制还是很多的，比如不能拖拽内部的小面板。未来的版本里，column 面板可能被删除。

## 参数

* panel

  面板对象构成的数组

-----------------------------
